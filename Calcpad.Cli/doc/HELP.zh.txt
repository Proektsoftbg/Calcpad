 命令:
————————————————————————————————————————————————————————————————————————————————
	NEW     - 清除当前会话并开始新会话
	OPEN    - 从磁盘打开存储的文件;
	SAVE    - 将当前文件保存到磁盘;
	LIST    - 列出磁盘上的可用文件;
	EXIT    - 关闭程序;
	RESET   - 清除输出并从头开始重新计算所有内容;
	CLS     - 清除输出并从最后一行继续;
	DEL     - 删除最后一行；
	DEG     - 将默认三角单位设置为度;
	RAD     - 将默认三角单位设置为弧度;
	GRA     - 将默认三角单位设置为等级;
	SETTINGS/OPTIONS -显示设置文件进行编辑; 
	LICENSE - 显示许可证文件;
	HELP    - 显示此帮助文件.

命令行选项（支持与 Calcpad.exe 相同的语言功能）：
———————————————————————————————————————————————————————————————————————————————— 
    calcpad input.cpd output.html - 计算 input.cpd 并导出
    calcpad input.cpd output.docx   将结果转换为相应的格式
    calcpad input.cpd output.pdf    （html、docx 或 pdf）；
		
    calcpad input.cpd html - 计算input.cpd并导出到input.html；
    calcpad input.cpd docx - 计算input.cpd并导出到input.docx；
    calcpad input.cpd pdf  - 计算input.cpd并导出到input.pdf；
	
    calcpad input.cpd      - 计算input.cpd并导出到input.html；.
	
	附加参数:
		-s   		   - 静默模式（不打开输出文件），例如:
		calcpad input.cpd output.html -s
